
### 第二幕：测试题（8题）

#### 焦虑维度模型（隐藏计分，用户不可见）

| 维度 | 代号 | 核心焦虑 |
|------|------|---------|
| 被替代焦虑 | R | 我的工作会不会没了 |
| 信息过载焦虑 | O | 学不完，根本学不完 |
| FOMO焦虑 | F | 别人都在用，我落后了 |
| 身份认同焦虑 | I | AI时代我算什么 |
| 行动瘫痪焦虑 | P | 知道该动但就是动不了 |

#### 题目

**Q1：周一早上看到一条新闻：「AI已经能做XX了」，你的第一反应？ 😰**
- A. 马上搜「怎么用」（+F2）
- B. 默默关掉，假装没看见（+P2）
- C. 转发朋友圈配文「完了」（+R2）
- D. 点进去看完，也就那样（低焦虑）

**Q2：同事用AI 20分钟搞定了你要做一天的方案 💀**
- A. 当场问他用的啥工具（+F2）
- B. 表面淡定，回家偷偷搜（+F1 +I1）
- C. 开始认真考虑转行（+R2）
- D. 太好了，抄一份给老板（低焦虑）

**Q3：你手机里有几个AI App？ 📲**
- A. 0个，在等它们证明自己（+P2）
- B. 1-2个，够用就行（低焦虑）
- C. 5个以上，常用的只有1个（+O2）
- D. 10个以上，大部分没打开过第二次（+O1 +P1）

**Q4：深夜刷到「35岁被裁，AI是主因」🌙**
- A. 算了算自己离35还有几年（+R2）
- B. 打开学习App，学了5分钟关了（+P2）
- C. 跟朋友吐槽到凌晨2点（+R1 +I1）
- D. 翻个身继续睡（低焦虑）

**Q5：有人说「你应该学学AI」，内心真实反应？ 🫠**
- A. 我知道啊！但从哪开始？（+P2）
- B. 我学了！但没学到点上（+O2）
- C. 不想学，但怕不学被淘汰（+R1 +I1）
- D. 已经在用了，挺好的（低焦虑）

**Q6：关注了多少「AI干货」账号？ 🔔**
- A. 0个，拒绝信息茧房（低焦虑）
- B. 几个，偶尔看看（低焦虑）
- C. 十几个，收藏几百条，实操0条（+O1 +P1）
- D. 多到记不清，天天被轰炸（+O2）

**Q7：如果AI能完全胜任你的工作？ 🤖**
- A. 终于能做真正想做的事了（低焦虑）
- B. 那我的价值在哪里？（+I2）
- C. 先焦虑三天，再假装没事（+P1 +R1）
- D. 立刻想「我还能做什么」（+F2）

**Q8：「AI焦虑」这件事本身…… 🌀**
- A. 贩卖焦虑，别上当（低焦虑）
- B. 焦虑正常，但得行动（低焦虑）
- C. 连焦虑都焦虑了，元焦虑（+O1 +I1 +P1）
- D. 我已经麻了（+P2）

### 第四幕：结果报告（4页）

#### 页面顺序

1. **雷达图** — 焦虑配方分析（五维雷达图 + 主要成分说明）
2. **冷知识** — 4条学术数据支撑（根据用户焦虑维度个性化匹配）
3. **结果卡片** — 焦虑指数 + 类型揭晓 + 截图金句 + 保存/再测按钮
4. **加米寄语** — 品牌收尾（根据焦虑程度分为两种版本）

#### 6种AI焦虑人格

| 人格类型 | 主维度 | 焦虑指数 | 判定条件 | 截图金句 |
|---------|--------|---------|---------|---------|
| **收藏夹战士** | O+P | 72 | 信息过载+行动瘫痪双高 | 收藏了一个亿，实操了个寂寞。 |
| **午夜焦虑体质** | R+I | 85 | 被替代+身份认同双高，R>I | 凌晨两点，AI新闻是我的安眠药反义词。 |
| **FOMO冲浪手** | F+O | 68 | FOMO主导 | 每个风口都想追，结果浪把我拍了。 |
| **佛系观望员** | P | 45 | 行动瘫痪突出(>4) | 人工智能千万条，与我无关第一条。 |
| **存在主义患者** | I+R | 90 | 被替代+身份认同双高，I≥R | 别人在学prompt，我在想人生意义。 |
| **已通关玩家** | 低焦虑 | 20 | 总分≤2 | AI是工具，我是甲方。 |

#### 计分逻辑

```javascript
// 累计五维得分
scores = {R, O, F, I, P}
tot = R + O + F + I + P

// 判定顺序
if (tot <= 2) → 已通关玩家
排序得分取 top1, top2：
if (top1∈{O,P} && top2∈{O,P}) → 收藏夹战士
if (top1∈{R,I} && top2∈{R,I}) → I≥R ? 存在主义患者 : 午夜焦虑体质
if (top1=F || (top1=O && top2=F)) → FOMO冲浪手
if (top1=P && P>4) → 佛系观望员
if (top1=I) → 存在主义患者
if (top1=R) → 午夜焦虑体质
fallback → 收藏夹战士
```

#### 加米寄语（两个版本）

**高焦虑版**（收藏夹战士/午夜焦虑体质/FOMO冲浪手/存在主义患者）：
> 看到这里，深呼吸一下。

## 四、插画风格：火柴人 SVG

### 风格关键词

Stick figure, bold lines, expressive poses, flat color background, playful, editorial illustration

### 设计规则

- **线条**：手绘感圆角线条，2.8px，深棕色 #2d2216
- **头部**：圆形，两个点眼睛 + 简单嘴巴线条
- **身体**：简笔线条，有动态姿势
- **道具**：每人一个标志性道具，道具用专属色彩色填充
- **动画**：SVG 画线动画（笔画逐渐出现）+ 道具弹入 + 浮动元素
- **实现**：纯 SVG 代码内嵌 HTML，矢量无损，支持 html2canvas 截图

### 6个角色设计

| 角色 | 专属色 | 姿势 | 道具 |
|------|--------|------|------|
| 收藏夹战士 | 琥珀黄 #F0AD4E | 双手各举手机，坐在书堆上 | 手机×2 + 收藏星星 |
| 午夜焦虑体质 | 靛蓝 #5C6BC0 | 裹被子举手机，惊恐表情 | 手机（发光）+ 闪电 |
| FOMO冲浪手 | 珊瑚红 #EF5350 | 踩冲浪板前倾，伸手够 | 冲浪板 + NEW标签 |
| 佛系观望员 | 薄荷绿 #66BB6A | 靠墙单脚站，闭眼微笑 | zzz + 口袋露手机角 |
| 存在主义患者 | 紫色 #AB47BC | 虚线身体，盘腿悬浮 | 问号×3 + 地面裂纹 |
| 已通关玩家 | 暖阳橙 #FFA726 | 站得稳，跟小机器人击掌 | 笔记本 + AI机器人 |
