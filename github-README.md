# 测测你的AI焦虑指数 🤯

一个基于学术研究的 AI 焦虑指数互动测试 H5 页面。

8 道场景化题目，测出你的 AI 焦虑类型：收藏夹战士、午夜焦虑体质、FOMO冲浪手、佛系观望员、存在主义患者、已通关玩家。

## 预览

> 部署后补充在线链接

## 特性

- 全屏滑动交互（Swiper.js）
- 6 种焦虑人格类型 + 五维雷达图
- 结果卡片一键保存为图片
- 纯前端，无后端依赖
- 移动端优先，桌面端自动适配（手机模拟框）

## 数据来源

- [Pew Research Center (2025)](https://www.pewresearch.org/) — AI and the Future of Workers
- Hajek et al. (2025) — AI Anxiety Scale, *PLOS One*
- Wang & Wang (2019) — AIAS 量表, *Interactive Learning Environments*

## 技术栈

- HTML / CSS / JavaScript（无框架）
- [Swiper 11](https://swiperjs.com/) — 全屏滑动
- [html2canvas](https://html2canvas.hertzen.com/) — 截图保存
- Google Fonts — Space Grotesk + Noto Sans SC

## 部署

纯静态文件，支持任意静态托管平台：

```bash
# Vercel
vercel deploy

# 或直接上传 index.html 到任意静态托管
```
